Installation

  1. 01 Install Fabric Loader for Minecraft 1.21.11 from fabricmc.net.
  2. 02 Download Fabric API and Cloth Config v17+ — both are required dependencies.
  3. 03 Download the latest Eymistaken's HUD .jar from Modrinth or GitHub Releases.
  4. 04 Drop all .jar files into your .minecraft/mods/ folder and launch the game.
  5. 05 Press Right Shift in-game to open the HUD Editor, or navigate to Options → Mods → Eymistaken's HUD → Settings.
// STRONGLY RECOMMENDED

Install Mod Menu

We strongly recommend installing Mod Menu alongside Eymistaken's HUD. Mod Menu adds a dedicated in-game mod list to the title screen, giving you quick access to the full Eymistaken's HUD settings panel at any time — including all module toggles, colors, positions, and scales — without needing to use the Right Shift shortcut.
