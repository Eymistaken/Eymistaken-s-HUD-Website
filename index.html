Core Features

7 Default Modules

Track everything that matters. Includes CPS, FPS, Ping, Combo, Keystrokes, Armor, and Reach trackers out of the box.

Keystrokes Designer

A completely interactive, precision designer for your Keystrokes. Snapping, context menus, drag-and-drop, and limitless configuration.

Modular API

Built for developers. Easily create and inject your own custom plugins and UI elements into the HUD rendering loop via the Eymistaken Hud Plugin framework.
