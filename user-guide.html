User Guide

Everything you need to set up, configure, and get the most out of every module in Eymistaken's HUD for Minecraft 1.21.11.

Accessing the HUD Editor

Hot Tip: Press Right Shift (RShift) from anywhere in-game to instantly open the HUD Editor.

The HUD Editor gives you a live preview of all active modules rendered on your screen. You can click and drag any element to reposition it anywhere — modules automatically snap to corners, edges, or the center using a smart stacking algorithm.

  • → Left-click + drag to move any module
  • → Scroll wheel to scale a module (50–300%)
  • → Right-click a module to open its settings context menu

The Keystrokes Designer

The crown jewel of Eymistaken's HUD. A fully interactive precision designer that lets you completely reimagine how your keystrokes look and feel — no external tools needed.

How to Use

  1. 01 Open the configuration menu and navigate to the Keystrokes section.
  2. 02 Click the Open Designer button in the Keystrokes context menu (right-click Keystrokes in the HUD Editor).
  3. 03 Drag and Drop: Left-click and drag any key panel to reposition it. Keys snap intelligently to alignment grids.
  4. 04 Context Menu: Right-click anywhere to add new keys, duplicate, or delete.
  5. 05 Edit Labels: Rename any key and use arrow keys for pixel-by-pixel text nudging.
  6. 06 Undo / Redo: Press Ctrl+Z to undo and Ctrl+Y to redo. History is saved automatically.

Context Menu Settings

Every module has a right-click context menu in the HUD Editor. Besides Reset Position, each module exposes its own settings:

CPS Module

  • → Enable CPS [ON/OFF]
  • → Text Color [color picker]
  • → Scale % [slider 50–300]
  • → Reset Settings

Keystrokes Module

  • → Enable Keystrokes [ON/OFF]
  • → Open Designer [action]

All other modules (FPS, Ping, Combo, Reach, Armor) are configurable via the Cloth Config settings menu, accessible from Mod Menu → Eymistaken's HUD → Settings.
